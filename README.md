# Cliffs Natural Resources (cliffs-natural-resources)

Cliffs Natural Resources is the legacy corporate name for what is now Cleveland-Cliffs Inc., a major mining and natural resources company that produces iron ore pellets primarily for steelmaking customers in North America. The company was renamed Cleveland-Cliffs in 2017 and the canonical profile lives at the [cleveland-cliffs](https://github.com/api-evangelist/cleveland-cliffs) repo. Cliffs Natural Resources / Cleveland-Cliffs does not publish a public developer portal or general-purpose REST API surface.
